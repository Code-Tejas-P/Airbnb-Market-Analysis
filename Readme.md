Airbnb Market Analysis Dashboard
================================

Tableau Data Visualization Project
----------------------------------

📊 Project Overview
-------------------

This project provides an interactive analysis of Airbnb listings, focusing on pricing trends, geographic distribution, and room type availability. The goal was to transform raw, messy listing data into actionable insights for property owners and travelers.

🛠️ Key Technical Challenges
----------------------------

*   **Data Cleaning:** Handled data type inconsistencies where the Price column was imported as a string with currency symbols ($) and commas.
    
*   **Calculated Fields:** Authored custom formulas to strip non-numeric characters and convert data to FLOAT for accurate aggregation.
    
*   **Dynamic Filtering:** Implemented dashboard actions that allow users to filter the entire view by selecting specific neighborhoods or room types.
